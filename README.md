### Overview

Amyotrophic Lateral Sclerosis (ALS) is a progressive neurodegenerative disorder that gradually impairs muscle function, including those involved in speech production. Subtle vocal tremors often emerge before severe clinical symptoms become evident, making speech a promising non-invasive biomarker for early diagnosis.

This research investigates whether speech tremors extracted from sustained phonation recordings can be effectively analyzed using machine learning techniques to support earlier detection of ALS and facilitate timely clinical intervention.



### Repository Purpose

This repository accompanies the published research and presents the methodology, machine learning workflow, and key findings of the study.
The implementation code used in the study is not included in this repository.



### Research Motivation

Early diagnosis of ALS remains challenging because initial symptoms are often subtle and difficult to distinguish from normal vocal variation or other neurological conditions.
Speech analysis provides a cost-effective, objective, and non-invasive approach for identifying early neuromuscular changes. By combining acoustic biomarkers with machine learning, this work explores an accessible framework for improving diagnostic support and enabling earlier intervention.



### Objectives

- Analyze sustained speech recordings for vocal tremors
- Extract clinically relevant acoustic biomarkers
- Evaluate machine learning models for ALS prediction
- Compare model performance using standard evaluation metrics
- Investigate speech as a scalable digital biomarker for early diagnosis

### Key Findings

The proposed machine learning framework successfully identified meaningful speech tremor characteristics associated with ALS.

Key findings include:

- Speech-derived acoustic features demonstrated strong diagnostic potential.
- Machine learning models effectively differentiated subtle speech abnormalities.
- The framework supports non-invasive and scalable screening approaches.
- Speech analysis shows promise as an objective tool for early ALS diagnosis.
